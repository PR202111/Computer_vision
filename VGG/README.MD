# VGG – Deep Convolutional Network

## Overview

This folder contains an implementation of the VGG architecture, known for its simplicity and depth using uniform 3×3 convolutional layers.

---

## Architecture Highlights

- Stacked 3×3 convolution layers
- Increased depth compared to AlexNet
- Max pooling for spatial downsampling
- Fully connected classifier

---

## Key Concepts Explored

- Impact of depth on representational power
- Small kernels with deeper networks
- Trade-off between simplicity and parameter count

---

## Observations

- Increasing depth improves feature abstraction
- VGG has a high parameter count
- Simple architectural design leads to strong performance
