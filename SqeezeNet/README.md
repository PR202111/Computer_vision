# SqueezeNet – Parameter-Efficient CNN

## Overview

SqueezeNet achieves AlexNet-level accuracy with significantly fewer parameters.

---

## Architecture Highlights

- Fire Modules
- 1×1 squeeze layers
- 1×1 and 3×3 expand layers
- Reduced parameter footprint

---

## Key Concepts Explored

- Model compression
- Efficient architecture design
- Trade-off between accuracy and size

---

## Observations

- High efficiency with fewer parameters
- Suitable for deployment scenarios
